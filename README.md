# Crime-Data-Analysis-Project

1 Introduction

This project focuses on analyzing Indian crime data using MS Excel to identify patterns, trends, and key insights across states, districts, and crime categories. The goal is to understand crime distribution and support data-driven decision-making.

2 Tools Used

MS Excel

3 Project Description

* Cleaned and prepared raw crime dataset for analysis
* Used Pivot Tables, Charts, and Conditional Formatting for visualization
* Performed state-wise, district-wise, year-wise, and category-wise analysis
* Answered multiple analytical questions to derive meaningful insights

4 Data Cleaning

* Corrected duplicate names such as A&N Island and A&N ISLAND
* Standardized inconsistent entries like D&N Island
* Removed total rows such as "zz total" and "Delhi UT total"
* Ensured proper formatting and consistency in dataset

5 Key Findings

* Maharashtra has the highest overall crime
* Patna is the top district in murder cases
* Theft is the most common crime category
* Year 2013 recorded the highest number of crimes
* Dacoity cases are decreasing, while theft and auto theft are increasing
* Crimes against women remain significantly high

6 Insights

* Urban and highly populated states show higher crime rates
* Dowry-related crimes and crimes against women remain serious concerns
* Year-over-year growth trends show increasing crime in states like Madhya Pradesh, Uttar Pradesh, and Maharashtra

7 Recommendations

* Enforce stricter laws in high-crime states
* Focus on women’s safety programs and policies
* Run awareness campaigns to reduce crime rates
* Use year-over-year analysis to identify crime hotspots early

8 Outcome

This project provides a clear understanding of crime trends and highlights critical areas that require attention. It demonstrates how Excel can be effectively used for data cleaning, analysis, and visualization.
